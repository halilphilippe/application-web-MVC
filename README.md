# Application de gestion des achats

Cette petite application web permet d'**enregistrer les achats** et de **voir les tendances**, notamment le **produit le plus acheté**.  
Elle est développée en **Python + Flask**, avec une logique simple et un test automatisé.

---

## Fonctionnalités

- Ajouter un achat (produit + quantité)  
- Consulter la liste des achats enregistrés  
- Voir le produit le plus acheté  
- Test automatisé pour valider la fonction `top_produit`
